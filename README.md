# Wizard World
# React + TypeScript + Vite

To run the project, clone the repo using
```
git clone https://github.com/k-vassilev/hitachi-wizard-world.git
```

Then run

```
npm install
```

And finally

```
npm run dev
```

The app should open on http://127.0.0.1:5173/
